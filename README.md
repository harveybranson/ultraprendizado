Engenharia de Prompts e "Cicatrizes": 1. Prompt para Meta-Aprendizagem e Mapeamento de Conteúdo
Este prompt ajuda a aplicar o primeiro princípio do Ultralearning, organizando o vasto conteúdo dos vídeos em uma estrutura lógica de aprendizado
.
"Atue como um mentor especialista em Python. Com base no vídeo [Inserir Link/Título do Vídeo], ajude-me a mapear o território deste conteúdo. Decomponha o que foi apresentado em três listas claras: 1. Conceitos (ideias que preciso entender profundamente, como lógica de herança ou decoradores); 2. Fatos (sintaxe, nomes de bibliotecas e comandos que preciso memorizar); e 3. Procedimentos (ações práticas como configurar o ambiente ou realizar o deploy). Para cada item, sugira uma forma de praticar que imite o uso real da habilidade, seguindo o princípio da Direção
."
2. Prompt para Intuição e Recuperação Ativa (Técnica de Feynman)
Utilize este prompt para garantir que você não está apenas memorizando superficialmente, mas desenvolvendo uma compreensão real dos conceitos complexos de Python
.
"Explique-me o conceito de [Inserir Tópico, ex: List Comprehensions ou Funções Assíncronas] apresentado no curso do [Mosh/AmigosCode]. Após a explicação, desafie-me a explicar o mesmo conceito de volta para você usando uma analogia simples para um leigo, sem usar termos técnicos. Avalie minha resposta com um feedback honesto e diga se minha explicação demonstra intuição profunda ou apenas memorização superficial. Se eu falhar, forneça um pequeno problema de código para eu resolver sem olhar a resposta
."
3. Prompt para Perfuração (Drilling) e Experimentação
Este prompt foca em atacar seus pontos fracos e ir além do básico, conforme os princípios 4 e 9 do método
.
"Identifiquei que meu 'gargalo' atual em Python é [Inserir Dificuldade, ex: Manipulação de dicionários aninhados]. Crie uma série de três exercícios de Perfuração isolados e crescentes em dificuldade para que eu possa praticar essa habilidade exaustivamente por meio de repetição deliberada. Após eu resolver, sugira um cenário de Experimentação onde eu deva aplicar esse mesmo conhecimento de uma forma não convencional ou utilizando uma biblioteca avançada (como Pandas ou NumPy), saindo do método padrão ensinado nos vídeos básicos
."
Esses prompts forçam a sua mente a lembrar (Recuperação) e a focar profundamente no que é difícil, acelerando o processo de se tornar um programador proficiente


Miniguia de Estudo (Entrega Final): Apresente o resultado final consolidado, que deve conter:
Resumos estruturados do assunto;
Um glossário com os principais conceitos aprendidos;
Um conjunto de prompts reutilizáveis que possam apoiar futuras revisões sobre o tema.
